What is frontend mentor, and how will it help me?

Frontend mentor offers realistic coding challenges to help developers improve their frontend coding skills with projects in html, css and javascript. It's suitable for all levels and ideal for portfolio building.

Is frontend mentor free?

The majority of challenges are free, yes.Frontend mentor do have some that are premium and require a pro subscription to access. It will say on each challenge whether they are free or premium, so it's easy to tell the difference.

Can i use frontend mentor projects in my portfolio?

Definitely! Feel free to use whatever you build in your portfolio. Helping developers add professional-looking projects to their portfolio was one of the reasons to create this platform!

How can i get help if i'm stuck on a challenge?

The best (and quickest) way to get help on a challenge is in frontend mentor discord server. There are thousands of other developers in there, so it's a great place to ask questions.Frontend mentor even have a dedicated "help" channel!
